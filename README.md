# 精通Rust_初学者指南2023中文版

* 2023.1.10

我的初衷并非是要翻译这本书，而是有感于近两年在学习和使用 Rust 时，总觉得自己还做不到丝滑般的编写 Rust 代码。仔细思考，问题出在我对 Rust 语言的基础掌握的还不够牢固。有鉴于此，我买了这本书。在学习本书的过程中，我不时地记笔记，随后转念一想，我何不完整的把此书翻译完呢？既可以查漏补缺，又可以分享给别人，可谓是一举两得。
